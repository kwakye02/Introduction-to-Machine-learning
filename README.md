# Introduction to Machine Learning

A beginner-friendly collection of machine learning notebooks and exercises. The repository introduces core machine learning ideas through practical Python examples.

## What This Covers

- Data preparation and exploration
- Supervised learning concepts
- Regression and classification workflows
- Model training, testing and evaluation
- Notebook-based machine learning practice

## How to Use

Open the notebooks in Jupyter Notebook, JupyterLab or Google Colab. Work through them in order, running each cell and reviewing the model outputs.

## Tools

Python, Jupyter Notebook, pandas, NumPy, matplotlib and scikit-learn.
